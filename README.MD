# How I Fix Your Code

Questo repository contiene implementazioni Python dei codici a correzione/rilevazione discussi nell'articolo [**How I Fix Your Code**]() pubblicato sul mio blog.

## Struttura

La cartella `part_1` contiene i codici visti nella **parte 1** dell'articolo.

## Requisiti

Questo progetto richiede `Python 3.11`.

## Esecuzione

Puoi eseguire gli esempi con:

```python
python part_1/repetition_code.py

python part_1/parity_check_1d.py

python part_1/parity_check_2d.py

```
