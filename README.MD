# How I Fix Your Code

Questo repository contiene implementazioni Python dei codici a rilevazione/correzione discussi nella serie **How I Fix Your Code** pubblicata sul mio blog.

- Parte 1: [How I Fix Your Code - 1](https://www.lucadivita.it/en/crittografia-sicurezza/how-i-fix-your-code-1/)
- Parte 2: [How I Fix Your Code - 2]()

## Struttura

- `part_1/`: codici trattati nella **parte 1** (ripetizione, parity check 1D e 2D).
- `part_2/`: codici trattati nella **parte 2** (Hamming code).

## Requisiti

Questo progetto richiede `Python 3.11`.

## Esecuzione

Puoi eseguire gli esempi con:

```bash
python part_1/repetition_code.py
python part_1/parity_check_1d.py
python part_1/parity_check_2d.py
python part_2/hamming_code.py
```
